# 配置与充值

## 创建 API Key

首先，登录并在[dashboard/apikey](https://www.codebyai.net/dashboard/api-keys)页面中创建专属 API Key。

![](/dashboard_apikey.png)

你可以根据需要对 API Key 的额度、过期时间进行设置。

![](/config_api_key.png)

点击按钮复制 API Key，后面配置会用到

![](/create_api_key.png)

## 充值

在 dashboard 下的[钱包](https://www.codebyai.net/dashboard/profile?topup=true)页面，选择合适的额度进行充值

![](/wallet.png)

本站与官方汇率 1:6，即在本站充值 $1，能获得等价于官方 $6 额度。
