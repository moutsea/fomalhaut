# Configuration & Top Up

## Create API Key

First, log in and create your exclusive API Key on the [dashboard/apikey](https://www.codebyai.net/dashboard/api-keys) page.

![](/dashboard_apikey.png)

You can set the quota and expiration time for the API Key as needed.

![](/config_api_key.png)

Click the button to copy the API Key, which will be used in later configuration.

![](/create_api_key.png)

## Top Up

On the [Wallet](https://www.codebyai.net/dashboard/profile?topup=true) page under the dashboard, select an appropriate amount to top up.

![](/wallet.png)

The exchange rate between this site and the official one is 1:6, meaning if you top up $1 here, you get a quota equivalent to $6 official value.
