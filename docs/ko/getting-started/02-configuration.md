# 구성 및 충전

## API Key 생성

먼저 로그인하고 [dashboard/apikey](https://www.codebyai.net/dashboard/api-keys) 페이지에서 전용 API Key를 생성하세요.

![](/dashboard_apikey.png)

필요에 따라 API Key의 할당량(quota)과 만료 시간을 설정할 수 있습니다.

![](/config_api_key.png)

버튼을 클릭하여 API Key를 복사하세요. 나중에 구성할 때 사용됩니다.

![](/create_api_key.png)

## 충전

대시보드 아래의 [지갑](https://www.codebyai.net/dashboard/profile?topup=true) 페이지에서 적절한 충전 금액을 선택하세요.

![](/wallet.png)

본 사이트와 공식 사이트의 환율은 1:6입니다. 즉, 여기서 $1를 충전하면 공식 가치로 $6에 해당하는 할당량을 얻을 수 있습니다.
