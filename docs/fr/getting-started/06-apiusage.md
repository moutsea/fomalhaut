# Aperçu de l'utilisation de l'API

### Tableau de bord

Vous pouvez facilement consulter l'utilisation globale des jetons de votre compte dans le tableau de bord.

![](/dashboard.png)

En même temps, vous pouvez également consulter les données d'utilisation détaillées pour chaque clé API, y compris le volume d'appels, la gestion des quotas et les paramètres d'expiration—tout cela en un coup d'œil.

![](/api_key_page.png)

### API Utilisation

Si vous utilisez une clé API partagée par quelqu'un d'autre, vous pouvez également consulter ses statistiques d'utilisation sur la page **API Utilisation** en haut de la barre de navigation.

![](/usage_page.png)
