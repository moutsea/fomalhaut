# Configuration et Rechargement

## Créer une clé API

Tout d'abord, connectez-vous et créez votre clé API exclusive sur la page [dashboard/apikey](https://www.codebyai.net/dashboard/api-keys).

![](/dashboard_apikey.png)

Vous pouvez définir le quota et la date d'expiration de la clé API selon vos besoins.

![](/config_api_key.png)

Cliquez sur le bouton pour copier la clé API, elle sera utilisée plus tard dans la configuration.

![](/create_api_key.png)

## Recharger

Sur la page [Portefeuille](https://www.codebyai.net/dashboard/profile?topup=true) du tableau de bord, sélectionnez le montant approprié pour recharger.

![](/wallet.png)

Le taux de change entre ce site et le site officiel est de 1:6, ce qui signifie que si vous rechargez 1 $ ici, vous obtenez un quota équivalent à 6 $ de valeur officielle.
